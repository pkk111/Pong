# Pong

This is the basic pong game with basic pixel graphics and sound effects.


# Controls
--------------
 * Laughting    : l L
 * Evil-Laught  : ;
 * Cruckle      : '

 * Player 1
   * Up    : w W
   * Down  : s S
   * Left  : a A
   * Right : d D
 * Player 2
   * Up     : Arrow_UP
   * Down   : Arrow_Down
   * Left   : Arrow_Left
   * Right  : Arrow_Right
